### 3.6 教材PDF下载指南（详细操作）

#### 方法一：tchMaterial-parser（推荐，最稳定）

1. **下载工具**：访问 https://github.com/happycola233/tchMaterial-parser/releases ，下载 `tchMaterial-parser-windows-x64.exe`（约16MB）
2. **获取accessToken**：
   - 浏览器打开 https://basic.smartedu.cn/tchMaterial 并登录
   - 按F12打开开发者工具，切换到Console
   - 粘贴以下代码并回车：
   ```javascript
   (function () {
     const authKey = Object.keys(localStorage).find((key) => key.startsWith("ND_UC_AUTH"));
     if (!authKey) { console.error("未找到Token，请先登录！"); return; }
     const tokenData = JSON.parse(localStorage.getItem(authKey));
     const accessToken = JSON.parse(tokenData.value).access_token;
     console.log("%cAccess Token: ", "color: green; font-weight: bold", accessToken);
   })();
   ```
   - 复制输出的accessToken
3. **运行工具**：双击exe，点击"设置Token"按钮粘贴保存
4. **下载教材**：粘贴以下URL到文本框中：
   - 生物七上：`https://basic.smartedu.cn/tchMaterial/detail?contentType=assets_document&contentId=405270eb-9f00-4bba-b0f6-da863340625f&catalogType=tchMaterial&subCatalog=tchMaterial`
   - 化学九上：`https://basic.smartedu.cn/tchMaterial/detail?contentType=assets_document&contentId=8c419b19-b3a9-4dbf-a8b4-546b7d337528&catalogType=tchMaterial&subCatalog=tchMaterial`
5. 点击"下载"，PDF自动保存

#### 方法二：Tampermonkey脚本（最便捷）

1. 安装Tampermonkey浏览器扩展
2. 安装脚本：访问 https://greasyfork.org/zh-CN/scripts/578069
3. 登录smartedu.cn后打开教材页面
4. 页面右下角出现"一键下载"按钮，点击即可

#### 方法三：直接浏览器下载

1. 登录 https://basic.smartedu.cn/tchMaterial
2. 打开教材详情页
3. 按F12→Network→筛选XHR→找到.pdf请求→右键复制URL
4. 在URL后添加 `?accessToken=你的accessToken值`
5. 浏览器直接访问该URL即可下载
